# Checklist Pengumpulan UAS

## Status Proyek

- [x] Source code aplikasi tersedia.
- [x] README tersedia.
- [x] `.env.example` tersedia.
- [x] `.env` tidak disertakan dalam paket.
- [x] Dokumentasi arsitektur tersedia.
- [x] Naskah video tersedia.
- [x] Template subtitle tersedia.
- [x] Unit test dasar berhasil dijalankan secara lokal.
- [ ] API key Google Gemini diisi oleh mahasiswa.
- [ ] API key LangSmith diisi oleh mahasiswa.
- [ ] Screenshot README diganti dengan screenshot aktual dari laptop mahasiswa bila diperlukan.
- [ ] Repository GitHub dibuat public.
- [ ] Video demo direkam dan diunggah ke Google Drive.

## Video

- [ ] Durasi 30-50 menit.
- [ ] Teori sekitar 15-25 menit.
- [ ] Menjelaskan LangChain.
- [ ] Menjelaskan LangGraph.
- [ ] Menjelaskan LangSmith.
- [ ] Menampilkan kode ketiga library.
- [ ] Menampilkan demo aplikasi.
- [ ] Menampilkan trace LangSmith.
- [ ] Subtitle terpasang dari awal sampai akhir.
- [ ] Video diunggah ke Google Drive, bukan YouTube.
- [ ] Akses link Google Drive dapat dibuka.

## GitHub

- [ ] Repository public.
- [x] Source code lengkap.
- [x] File `.env` tidak ikut diunggah.
- [x] `.env.example` tersedia.
- [x] README berisi deskripsi proyek.
- [x] README berisi screenshot atau preview aplikasi.
- [x] README berisi cara instalasi dan menjalankan.
- [ ] Repository dapat di-clone dan dijalankan pada laptop mahasiswa setelah API key diisi.

## Sebelum Dikirim

- [ ] Uji link video dengan browser incognito.
- [ ] Uji link GitHub dengan browser incognito.
- [ ] Pastikan tidak ada API key di video, kode, commit, atau screenshot.
- [ ] Kirim kedua link melalui Google Classroom.
