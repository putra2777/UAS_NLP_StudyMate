Natural Language Processing (NLP) adalah bidang yang mempelajari bagaimana komputer memproses, memahami, dan menghasilkan bahasa manusia. Dalam sistem modern, NLP banyak menggunakan model bahasa besar atau Large Language Model (LLM). Model tersebut mempelajari pola statistik dari data teks dalam jumlah besar sehingga mampu melakukan peringkasan, klasifikasi, tanya jawab, ekstraksi informasi, dan generasi teks.

LangChain merupakan framework yang menyediakan komponen untuk menghubungkan model bahasa dengan prompt, parser, retriever, memory, dan tools. Komponen-komponen tersebut dapat disusun sebagai chain sehingga alur pemrosesan menjadi modular. Pada proyek ini, LangChain digunakan untuk membentuk prompt analisis, prompt peringkasan, prompt pembuatan kuis, serta prompt evaluasi kualitas.

LangGraph merupakan framework orkestrasi untuk membangun workflow berbasis graph. Setiap langkah direpresentasikan sebagai node, sedangkan perpindahan antar-langkah direpresentasikan sebagai edge. State digunakan untuk menyimpan data yang dibutuhkan sepanjang workflow. Conditional edge memungkinkan sistem memilih jalur berdasarkan kondisi tertentu. Pada proyek ini, hasil review diberi skor. Apabila skor belum mencapai threshold, workflow masuk ke node revisi dan kembali ke node review.

LangSmith menyediakan observabilitas terhadap aplikasi LLM. Setiap pemanggilan model dan setiap tahap workflow dapat dilacak dalam bentuk trace. Pengembang dapat memeriksa input, output, durasi, error, metadata, dan urutan proses. Informasi tersebut membantu proses debugging, evaluasi, dan peningkatan kualitas aplikasi.

Kombinasi LangChain, LangGraph, dan LangSmith menghasilkan pemisahan tanggung jawab yang jelas. LangChain menangani komponen pemrosesan LLM, LangGraph menangani orkestrasi alur, dan LangSmith menangani pemantauan serta evaluasi proses. Arsitektur ini lebih mudah dikembangkan dibandingkan satu fungsi besar yang menjalankan seluruh proses secara linear tanpa state dan observabilitas.
